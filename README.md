# OSDM-CustomMedia
aka: OSDM:CM for short, is a project that searches to bring extra resources to users of OSDM. The project features extra resources like the following:
- Animations
- Coppers
- Fonts
- Layers
- Logos
- Miscellaneous
- Modules
- Prefs
- Textures
and more (Extras)...
